### Installation

\_Below are instructions to get started

1. Clone the repo
   ```sh
   git clone https://github.com/ac12644/Fundraising-Dapp.git
   ```
2. Install packages
   ```sh
   yarn
   ```
3. Run application
   ```sh
   yarn run dev
   ```
